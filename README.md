# AFrame-SeeClarke
> Move around your A-Frame environments hands/device free via your webcam!

## Development
``` bash
# Setup
git clone https://github.com/labofoz/aframe-seeclarke
cd aframe-seeclarke
yarn

# Commands
yarn build
yarn dev  # With hotreloading
yarn test
```
